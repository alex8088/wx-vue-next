## Features

- 🍥 Base on WeUI Design
- 🚀 More then 40 components（WIP）
- 📗 Extensive documentation and demos
- 🔥 Written in TypeScript
- 🔥 Support Vue 3
- 💪 Support on-demand import
- 💪 Support dark mode
- 💪 Support custom theme
