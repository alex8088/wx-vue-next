# wx-vue

> A Vue.js 3 Mobile UI components library based on the WeUI Design

---

## Features

- 🍥 Base on WeUI Design
- 🚀 More then 40 components（WIP）
- 📗 Extensive documentation and demos
- 🔥 Written in TypeScript
- 🔥 Support Vue 3
- 💪 Support on-demand import
- 💪 Support dark mode
- 💪 Support custom theme
